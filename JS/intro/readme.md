## In programming

** *variables* are *identifiers* used to store data in a memory location that can be re-used in program execution.
identifiers are uique label assigned to the variable.

**Classification of data types**

1. ***Primitive Data Types***
-   Number: Handles both whole integers and decimal fractions ie 42 ,3.142

-   String: Alphanumeric text wrapped in single, double  "Hello"

-   Boolean: Binary logic states evaluating only to `true` or `false`.
-   Undefined: The default state of a variable that has been declared but not 

       yet given a value (e.g., `let x;`).
-   Null: An intentional, explicitly assigned marker representing an empty value

-   BigInt: Used for massive integers that exceed the safe calculation limits of the standard `Number` type.
-   Symbol: A completely unique, hidden, and immutable identifier often used as unique keys for object properties. 

2. Non-Primitive (Reference) Data Types
Non-primitive structures store memory pointers pointing to where the actual data resides. 
-   Object: Collections of key-value pairs used to model real-world entities (e.g., `{ name: "Alice", age: 25 }`).
-   Array: An ordered list of values mapped by numerical indexes (e.g., `[1, 2, 3]`).
-   Function: Reusable blocks of executable code that can be stored inside variable
